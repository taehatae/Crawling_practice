# 비정형데이터 AI분석

# 중간고사 팀 프로젝트

### 팀명 : 기름기 쫙 뺀팀
|황태하|신형진|

### 키워드 : '호르무즈 해협'

### 데이터 수집 기간 설정 : 갈등 최고조 및 전쟁 발발 기간으로 설정
### (2026.02.01 ~ 2026.04.10)

### 사용된 분석 기법
- 키워드 빈도수 측정 후 그래프로 시각화 : 호르무즈 해협 이슈에서 사람들이 가장 주목한 핵심 키워드가 무엇인지 파악
- 워드 클라우드로 시각화 : 단어의 크기와 색상으로 시각적으로 더 한눈에 파악
- TF-IDF : 단순히 자주 등장하는 단어가 아니라, "다른 글에서는 안 나오는데 이 글(또는 이 그룹)에서만 유독 강조되는 진짜 핵심어" 파악
- K-Means : 사람들이 호르무즈 해협 사건의 어떤 측면에 가장 민감하게 반응하고 있는가 파악
